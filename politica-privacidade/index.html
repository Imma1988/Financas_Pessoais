Política de Privacidade

Última atualização: 28 de junho de 2026  ·  Versão 1.0

Resumo simples: o MoneyHolic guarda os teus dados na tua própria conta Google Drive. Nós não vemos, não armazenamos nem vendemos qualquer informação financeira tua. O único dado que recebemos do Google é o teu endereço de e-mail, utilizado exclusivamente para identificar a tua conta.

1. Quem somos

O MoneyHolic (moneyholic.pt) é uma aplicação de gestão de finanças pessoais desenvolvida e operada em Portugal.

Contacto: info@moneyholic.pt

2. Que dados recolhemos

2.1 Dados fornecidos pelo Google (OAuth 2.0)

Quando te autenticas com a tua conta Google, o MoneyHolic recebe, com o teu consentimento explícito, as seguintes informações:

  • Endereço de e-mail — utilizado apenas para identificar a tua sessão
  • Nome de perfil (opcional) — pode ser apresentado na interface da aplicação
  • Foto de perfil (opcional) — apresentada no avatar da interface

Estes dados são obtidos através do scope openid email profile do Google Identity e não são transmitidos a terceiros nem armazenados nos nossos servidores.

2.2 Dados do Google Drive

O MoneyHolic solicita acesso ao teu Google Drive (https://www.googleapis.com/auth/drive) exclusivamente para:

  • Criar, ler e atualizar um ficheiro JSON (financas_pessoais.json) na tua conta Google Drive, onde são guardados os teus registos financeiros (movimentos, cofres, metas, configurações)
  • Ler ficheiros partilhados por ti (p. ex., ficheiro de dados de um cônjuge ou parceiro), apenas mediante indicação explícita por tua parte

Os dados financeiros residem exclusivamente na tua conta Google Drive. O MoneyHolic não copia, não transmite nem armazena esses dados em qualquer servidor externo.

2.3 Dados de importação de extratos (IA)

Quando utilizas a funcionalidade de importação de extratos bancários, o conteúdo do ficheiro (PDF ou CSV) é enviado diretamente para a API da Google Gemini para processamento de inteligência artificial. Este envio é feito entre o teu browser e a API da Google. O MoneyHolic não guarda nem acede ao conteúdo desses ficheiros.

Consulta a Política de Privacidade da Google para detalhes sobre o tratamento de dados pela API Gemini.

2.4 Dados de utilização

O MoneyHolic não utiliza ferramentas de analytics de terceiros nem recolhe dados de comportamento de utilização.

3. Base legal para o tratamento (RGPD)

Ao abrigo do Regulamento Geral sobre a Proteção de Dados (RGPD – Regulamento UE 2016/679), o tratamento dos teus dados baseia-se em:

  • Consentimento (Art. 6.º, n.º 1, al. a): autorização explícita que concedes ao iniciar sessão com o Google e ao aprovar as permissões de acesso ao Drive
  • Execução de contrato (Art. 6.º, n.º 1, al. b): necessidade de aceder ao Drive para prestar o serviço que solicitaste

4. Como utilizamos os dados

  • Identificar a tua conta e personalizar a interface
  • Ler e guardar os teus dados financeiros no teu Google Drive
  • Processar extratos bancários via IA (Gemini), enviando o ficheiro diretamente do teu browser para a API da Google

Não utilizamos os teus dados para publicidade, perfilagem ou qualquer finalidade diferente da prestação do serviço MoneyHolic.

5. Com quem partilhamos dados

O MoneyHolic não vende, não aluga nem partilha os teus dados pessoais com terceiros, com a seguinte exceção técnica:

  • Google LLC — fornecedor de autenticação (Google OAuth), armazenamento (Google Drive) e IA (API Gemini). As transferências para os servidores da Google nos EUA são cobertas pelas Cláusulas Contratuais Tipo aprovadas pela Comissão Europeia

6. Retenção de dados

Os dados armazenados no Google Drive são controlados por ti e persistem enquanto mantiveres o ficheiro na tua conta. Podes eliminá-los a qualquer momento através do próprio Google Drive.

O MoneyHolic não mantém cópias dos teus dados fora do Google Drive.

7. Os teus direitos (RGPD)

Como titular dos dados, tens os seguintes direitos, exercíveis através do contacto indicado abaixo:

  • Acesso — saber quais os dados que tratamos
  • Retificação — corrigir dados incorretos
  • Eliminação — solicitar a eliminação dos dados ("direito a ser esquecido")
  • Portabilidade — receber os dados em formato estruturado
  • Oposição — opor-te ao tratamento em determinadas circunstâncias
  • Retirada do consentimento — a qualquer momento, sem afetar a licitude do tratamento anterior

Para revogar o acesso do MoneyHolic ao teu Google Drive, acede a myaccount.google.com/permissions e remove a autorização à aplicação.

Tens ainda o direito de apresentar reclamação à CNPD – Comissão Nacional de Proteção de Dados (autoridade de controlo em Portugal).

8. Segurança

O MoneyHolic utiliza HTTPS em todas as comunicações. Os tokens de acesso OAuth são armazenados apenas em memória durante a sessão (não são persistidos em localStorage nem cookies). O acesso ao Google Drive é limitado ao ficheiro criado pela aplicação.

9. Cookies

O MoneyHolic não utiliza cookies de rastreamento nem cookies de terceiros. Poderão ser utilizados cookies técnicos estritamente necessários para o funcionamento da autenticação Google (geridos pelo próprio Google).

10. Alterações a esta política

Sempre que esta política for atualizada de forma significativa, a data de "última atualização" será alterada e, sempre que possível, será enviada uma notificação por e-mail. O uso continuado do serviço após a atualização constitui aceitação das alterações.
